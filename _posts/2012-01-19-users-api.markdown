Users API
=========

### Retrieve Confirmed Agents of a company

#### GET /users.json

*Retrieves all confirmed agents of the company*

#### Status Codes

Success: 200

#### Example

GET /users.json

  "users":[
    {

*Retrieves info about the user with id _id_*
